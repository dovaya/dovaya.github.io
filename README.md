# dovaya.github.io

Uses [Dovaya's Hugo Theme](https://github.com/dovaya/dovaya-hugo-theme).

Installing:

```
git submodule add --depth=1 https://github.com/dovaya/dovaya-hugo-theme.git themes/dovaya-hugo-theme
git submodule update --init --recursive
```
Updating:

```
git submodule update --remote --merge
```s
